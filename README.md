Mumbai Crime Data Analysis (September 2025)

This project analyzes **Mumbai Police Crime Data (September 2025)** obtained from official reports.  (https://mumbaipolice.gov.in)
The goal is to understand crime trends, detection efficiency, and patterns across different categories such as IPC crimes, Women-related crimes, NDPS (drugs), Brothels, Economic Offences, and Cyber Crimes.

The project includes:
- Converting the datasets in pdf format to excel using the website https://online2pdf.com
- Cleaning data in the excel file, adjusting columns to make the data easy for analysis.
- Data cleaning using Pandas.
- Exploratory Data Analysis using Pandas.
- Visualization using Matplotlib and Seaborn.
- Connecting and loading data into MySQL database.
- Running some queries in MySQL.

The dataset consists of **6 sheets**, each representing a major crime category:
1. **IPC** – Indian Penal Code crimes  
2. **Women** – Crimes against women  
3. **NDPS** – Drug and narcotics offences  
4. **Brothels** – Human trafficking and prostitution-related offences  
5. **Economic Offences** – Financial fraud and forgery  
6. **Cyber** – Cyber crimes such as phishing and online fraud  


Tools & Technologies:
- **Python** (Pandas, NumPy, Matplotlib, Seaborn)
- **MySQL** (data storage and analytical queries)
- **Google Colab / Jupyter Notebook**
- **GitHub** (project versioning)
